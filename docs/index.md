# @spore-ui/tskit

工具函数库 / 面向 TypeScript

## 何时使用

本工具库意图在日常业务积累中，留存各种常见工具函数，解决方案，Hack方案。

工具函数进行了分门别类的梳理，推荐使用时引用确定版本，直接引用函数，可以直接减少代码打包体积。

## 快速上手

```shell
npm i @spore-ui/tskit
```

```javascript
// 引入一个方法
import { delay } from '@spore-ui/tskit';
```
